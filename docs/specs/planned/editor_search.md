# Program検索仕様

> Status: Draft

## 目的

本書はProgram Editorの検索機能の責務と実装時期を定義する。

---

# 実装時期

Program検索はPhase 2の対象外とする。

検索機能を実装するPhaseを決定する際に、本書へ検索文字列の正規化、部分一致規則、結果順、検索対象フィールド、キーボード操作を追記してから実装する。

---

# 責務境界

検索はProgramを読み取り、該当Node IDの一覧を返す。検索によってProgramを変更しない。

検索結果の選択によるNode選択やキャンバス位置の移動はUI状態だけを変更し、UndoとRedoの履歴へ追加しない。

最低限の検索対象候補は次のとおりとする。

- 命令種別
- 名前
- コメント
- ラベル

「名前」と「ラベル」が参照する具体的なProgramフィールドは現行Programモデルで定義されていないため、Programモデルを拡張せずに推測して実装してはならない。
